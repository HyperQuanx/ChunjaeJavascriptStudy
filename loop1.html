• for 문

    for ( 초기값; 조건; 증가식 ) { 실행문; 조건문; 반복문; 함수; ...}

  • 초기값 : for 문에서 사용하는 반복 조건값을 초기화
  • 조건 : 조건이 true 동안에 for 문이 반복
  • 증가식 : for문이 반복되는 동안 초기값 증가
